# US Data Analyst Job Market Analysis
## Overview

This project aims to analyze the job market in the United States for Data Analysts in order to identify trends in salary, skill demand, and geographic distribution of opportunities. Using job posting data, I explored which technical skills are most frequently requested, how salaries vary across skills and locations, and where the greatest concentration of data analyst positions exists. 
## Objectives
* Identify the most sought-after technical skills for Data Analyst positions.
* Analyze average salaries across different technical skill sets.
* Examine salary distributions across states.
* Determine which states have the highest concentration of Data Analyst job opportunities.
* Develop an interactive dashboard to communicate findings effectively.

## Key Findings
* Most Requested Skills
SQL was the most frequently requested technical skill. Python and Tableau were also highly demanded across job postings.
* Salary Analysis
Python was associated with the highest average salary among the analyzed skills.
Salary differences between skills were relatively modest, suggesting employers value a combination of these skills.
* Geographic Trends
California had the highest number of Data Analyst job postings.
New York and Texas were also major hubs for Data Analyst opportunities.
Significant variation exists in salary distributions across states.
